# 🇬🇧 RDVS Numériques DuMAS - 5 mars 2026<br>
# "Deap Learning: a simple Dense Neural Network to classify the MNIST images and a Convolutional Neural Network to classify micrographs."
This repository contains the documents from the second part of the "RDVS Numériques DuMAS" day on June 5, 2026, at ENSAM in Talence, devoted to Deep Learning.<br>

To use the notebooks:
- Install the Python project manager __uv__ (see https://docs.astral.sh/uv/getting-started/installation/)
- Download the zip from the GitHub repository (green [Code v] button at the top of the page)
- Unzip the archive and, in a terminal (Linux/macOS terminal or Windows PowerShell), go to the `Micrographs_classif_at_I2M` folder using the command:<br>
`cd <path to>/Micrographs_classif_at_I2M>`<br>
where `<path to>/Micrographs_classif_at_I2M` is the path to the `Micrographs_classif_at_I2M` folder, which you can retrieve using your file browser.
- Type the command `uv sync` to install the Python modules in the Python Virtual Environment of the `Micrographs_classif_at_I2M` folder.
- Launch Jupyter by typing: `uv run jupyter lab` or marimo by typing: `uv run marimo edit`.
